### Bejeweled-RL-Trainer

How to run:
1. Download the repository and download all necessary dependencies
2. Open your copy of Bejeweled 3 and navigate to Zen Mode
3. Run main.py
4. In the GUI, ensure that the captured window is "Bejeweled 3"
5. Open Options and make any necessary changes
6. Start training or load your own training model

This project is still in progress and many features are still incomplete. You also must own a copy of Bejeweled 3.
